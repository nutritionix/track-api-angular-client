nixTrackCalculator
service in module nix.track-api-client

Description

Used to calculate recommended calories number

Methods

  • calculateBmr(gender, weight, height, age)

    Calculate basal metabolic rate
    Men BMR = 88.362 + (13.397 x weight in kg) + (4.799 x height in cm) - (5.677 x age in years)
    Women BMR = 447.593 + (9.247 x weight in kg) + (3.098 x height in cm) - (4.330 x age in years)

    Parameters
    ParamTypeDetails
    genderstring

    "f" for female, "m" for male

    weightint

    Person weight in kilograms

    heightint

    Person height in centimeters

    ageint

    Person age in years

    Returns
    number

    Basal metabolic rate

  • calculateRecommendedCalories(gender, weight, height, age, exerciseLevel)

    Calculate recommended number of calories for particular person

    Parameters
    ParamTypeDetails
    genderstring

    "f" for female, "m" for male

    weightint

    Person weight in kilograms

    heightint

    Person height in centimeters

    ageint

    Person age in years

    exerciseLevelint

    Index from exerciseLevels array. 0-4, defaults to 0 = Little to no exercise

    Returns
    number

    Recommended number of calories

Properties

  • exerciseLevels

    List of exercise levels to factor BMR

    [0] Little to no exercise
    [1] Light exercise (1-3 days per week)
    [2] Moderate exercise (3-5 days per week)
    [3] Heavy exercise (6-7 days per week)
    [4] Very heavy exercise (twice per day, extra heavy workouts)
